# House Price Prediction using Modified XGBoost

This project predicts house prices using a machine learning model based on Modified Extreme Gradient Boosting (XGBoost).

## Overview
The goal of this project is to improve house price prediction accuracy by handling complex and changing market data more effectively than traditional models.

## Technologies Used
- Python
- Machine Learning
- XGBoost
- Pandas
- NumPy
- Scikit-learn

## Project Type
Academic / College Project
